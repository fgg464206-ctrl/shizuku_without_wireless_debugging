# LaiLightSMP Tool (Shizuku Helper) 🚀

This tool is designed to help you start the Shizuku service easily via an OTG cable without any hassle! ✨

## 🛠 How to Use

1. Open Termux on your phone.

2. Copy and paste the following command into your Termux terminal and press Enter:

   curl -sL https://raw.githubusercontent.com/fgg464206-ctrl/shizuku_without_wireless_debugging/refs/heads/main/install.sh | bash

3. Once the installation is complete, run the tool by typing:

   ./shizuku_without_wireless_debugging.sh

4. Choose your preferred option from the menu! 📱
